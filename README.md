# BFS_Visualization
Breadth First Search ( in a Graph ) Visualization using Jquery

## Features:
- Drag to draw walls
- Selectable Starting and ending points
- Clearing visited nodes & path ( not resetting the walls )
- Reseting the playground ( erases everything )
